## Course Expectations

<div>

IT-CS 115 (and 142 and 143) teach you foundational skills.  You\'ll need these skills to do everything from writing code to maintaining code to making sense of code that AI has given you to fixing and integrating code with an AI.

</div>

<div>

Because of that, the exams will be **paper-and-pencil exams**.  This both ensures that the instructor is testing what **you** know and it\'s good preparation for future, whiteboard-based, technical interviews.

</div>

## How to use AI productively

-   Use it to develop your knowledge, but always remember that you\'re going to need to have this foundational material in your head and at your fingertips in order to be productive at coding.
-   [Definitely use AI to explain stuff to you - you can paste code into it and then ask it questions about the code (\"Why is it necessary to start this variable at 1 instead of 0?\", \"What does this function do?\", etc).]{style="color: var(--ic-brand-font-color-dark); font-family: inherit; font-size: 1rem;"}
    -   [Remember that it\'s not perfect, so if the answer really doesn\'t make sense it may be that it\'s wrong.]{style="color: var(--ic-brand-font-color-dark); font-family: inherit; font-size: 1rem;"}
-   [Use AI to help you get unstuck when you\'re really stumped.]{style="color: var(--ic-brand-font-color-dark); font-family: inherit; font-size: 1rem;"}
    -   [At the same time, if you instantly ask AI for help every time something doesn\'t work then you\'ll never learn how to do it yourself.]{style="color: var(--ic-brand-font-color-dark); font-family: inherit; font-size: 1rem;"}
    -   [A good rule of thumb is this: spend 10 minutes trying to get something to work before asking for help (from other people, or AI, or the Internet, etc)]{style="color: var(--ic-brand-font-color-dark); font-family: inherit; font-size: 1rem;"}
