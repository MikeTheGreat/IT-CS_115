## **Tasks**

**NOTE: You must limit yourself to the topics and techniques we have covered in this class (or a pre-req class), and within this class you can only use those topics that we were covered when this assignment was first due.**

Write a program using the TinkerCAD simulator and/or the Arduino IDE that will animate the LEDs that your combined team (\"combined team\" meaning \"IT-CS students *and* ART students\") needs in order to produce the plexiglass artwork that you\'ve all decided on.

In order to get full marks for this IT-CS 115 assignment your final project will need to include the following elements:

-   The program must run indefinitely (as long as the Arduino is plugged in & powered on).\
    (Mostly this means that your loop() function should keep running, over and over, while it\'s plugged in.  Make sure it doesn\'t crash or stop at any point).
-   4 (or more) LEDs 
-   At least 2 separate circuits (i.e., you can\'t put all 4 LEDs on the same pair of wires)
-   Your program must light up (at least) 2 LEDs at the same time, at least once in the program (after that you can light up or not light up whatever combination you want, including only lighting up single LEDs)
-   Your program must light up an LED in a sequence of varying lengths of time  (for example, having an LED light up for a long-short-short-short sequence - maybe 1 second followed by 3 flashes of 1/10 of a second)

NOTE: Anywhere this assignment says \"LED\" you need to connect the LED with separate wires (and a resistor!  Don\'t forget the resistor!) to the Arduino.  This does NOT include the built-in LED on the Arduino board itself, even if that LED gets incorporated into the artwork.  So when you see a requirement that says (for example) \"4 (or more) LEDs\", this means that you\'ll need to get 4 LEDs (separate from the one on the Arduino board), connect them to resistors and wires, etc.
