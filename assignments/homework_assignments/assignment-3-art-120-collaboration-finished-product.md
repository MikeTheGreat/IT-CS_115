## You ARE allowed to work in groups this assignment
For this assignment, you should work with the other students in your group.  If you have questions, please contact the instructor. 


## Purpose

For this assignment you will finish the Art project by creating the working, finished product - a physical, real-world Arduino that fulfills the requirements listed in \"[Assignment 3 - ART 120 Collaboration - TinkerCAD design](./assignment-3-art-120-collaboration-tinkercad-design.md "Assignment 3 - ART 120 Collaboration - TinkerCAD design")\", and that works with one of the 3D printed designs from the Art students in your team.

 
## **Tasks**

**NOTE: You must limit yourself to the topics and techniques we have covered in this class (or a pre-req class), and within this class you can only use those topics that we were covered when this assignment was first due.**

### Create the finished, working, art project

You must create a TinkerCAD design for the current version of the project (this means that if the Art design has changed since the last assignment for this project then you must update the TinkerCAD design to it accurately reflects the physical Arduino that you produce.

You must create the the working, finished product - a physical, real-world Arduino that fulfills the requirements listed in \"[Assignment 3 - ART 120 Collaboration - TinkerCAD design](./assignment-3-art-120-collaboration-tinkercad-design.md "Assignment 3 - ART 120 Collaboration - TinkerCAD design")\", and that works with one of the 3D printed designs from the Art students in your team.

 

### Document the working, physical artwork

In order to show that your Arduino is working as part of the overall art project you must upload a video showing the art project along with your flashing LEDs.  The easiest way to do this will be to record a short (10-30 second) video using your phone, and to then upload the resulting video file to Canvas.

## **Submission**

Submit the following:

1.  A screenshot of the current, simulated Arduino in TinkerCAD (do this even if the design hasn\'t changed since the last assignment)
2.  The source code file for your program file
3.  Documentation of your team\'s interactions
4.  A video showing the physical artwork working\
    If the video is too large for Canvas then you should upload is somewhere else (such as Google Drive, or MS OneDrive, or DropBox, etc) and then include a link to that video.  Make sure that the link is public!  (In other words, make sure that anyone can open that link without needing special permissions)

 

## **Grading Criteria**

Your work will be graded using the attached rubric.

Remember that you must make the Arduino actually work as part of the art project.  When the Arduino is plugged in then the program must start and run, the LEDs need to light up, and this needs to continue running until the Arduino is unplugged.
